## Alert

This component can be used to highlight important information.

```md
<Alert color="blue" title="Title">
  This is an alert.
</Alert>
```

Props:
- title: `string`, `required`
- color: `Color`, `required`

Available `Color` values:
- Orange
- Red
- Green
- Blue
- Yellow
